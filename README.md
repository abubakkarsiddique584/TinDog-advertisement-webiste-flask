# 🐶 TinDog Advertisement Website (Flask Project)

## 📌 Project Overview
TinDog is a simple startup landing page website built using Flask, HTML, and CSS. It is a fictional Tinder-style platform for dogs where users can discover and match with nearby dogs.

This project demonstrates frontend design skills combined with basic Flask backend routing to serve a responsive website.

---

## 📁 Repository Contents
- `server.py` → Flask backend server to run the website  
- `index.html` → Main landing page (frontend UI)  
- `style.css` → Custom styling for the website  
- `dog-img.jpg` → Hero image asset  
- `iphone.png` → App showcase image  
- `bizinsider.png`, `mashable.png`, `techcrunch.png`, `tnw.png` → Partner/logo assets

- Responsive landing page design 📱💻
